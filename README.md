# წინამძღვრიანთკარი Registry Tracker

Scrapes land registry data for სოფელი წინამძღვრიანთკარი from
naprweb.reestri.gov.ge and displays it via Streamlit.

## Running locally

    pip install -r requirements.txt
    python scraper.py --mode daily   # fetch latest
    streamlit run dashboard.py       # open dashboard
